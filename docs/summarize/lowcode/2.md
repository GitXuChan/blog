# No.2 闪搭的各个功能菜单

# <H2Icon />应用首页

![闪搭应用首页](https://nocode.cloudred.cn/help/_images/%E9%A6%96%E9%A1%B5%E9%A1%B5%E9%9D%A2.png)

- 侧标栏：位于页面右上角，用于控制系统菜单边栏的打开和关闭

- 首页：点击可快速跳转回首页

- 应用：用于展示该系统所有项目，点击项目卡片，可进入对应项目

- 审批：点击可快速跳转审批页面

- 通讯录：点击可快速跳转通讯录页面

- 消息：点击可快速跳转到消息页面

- 设置：点击，可下拉选择应用编辑，系统配置，系统日志，设置当前页为首页，语言选择，清除缓存等操作

- 帮助：可点击进入帮助中心或查看系统版本

- 登录人：用于展示人员姓名并可快速跳转到我的详情页面以及退出系统

- 搜索框：用于快速搜索你需要的应用

- 视图切换按钮：可在列表视图和图标视图中进行切换
